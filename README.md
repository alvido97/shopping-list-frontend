# shoppinglist

## Project setup
```
npm install
```

## Setup .env file 
```
Create new file called .env

```

## Put the variable VUE_APP_API_URL with your url
```
VUE_APP_API_URL=your-host.com/api
example - 
VUE_APP_API_URL=https://shopping-list.test/api

```

### Compiles and hot-reloads for development
```
npm run serve
```
